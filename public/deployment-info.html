Database Configuration Required

Your Ignyt application is deployed, but it requires a connection to a PostgreSQL database to function properly.

Please complete these steps:

  1. Go to your Replit Dashboard
  2. Access the Deployment settings for this app
  3. Add DATABASE_URL to the Secrets section
  4. Use the same PostgreSQL connection string from your development environment
  5. Redeploy the application
